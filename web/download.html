The Comma project has not yet released any files, as the system is still in the planning stage. If you are interested in helping to develop Comma, please subscribe to the developers mailing list.

The development sources can be fetched through Git using: 

git clone git://git.savannah.nongnu.org/comma.git

You can also browse the source code online.
