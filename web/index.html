Comma Overview

The Comma project is currently in the planning stage. Therefore, little is available at this time. Development is hosted on Savannah.

This principle aim of this project is to provide both a compiler implementation and a language specification. The language is influenced most notably by SPAD (the library compiler for the Axiom computer algebra system), and Aldor. Language features are focused on providing facilities which are well suited for implementing generic mathematical algorithms.
